Installation instructions

COBRA is an open source finite element solver that is a modified version of MOOSE, an open source finite element framework developed at Idaho National Laboratory. The MOOSE solver is based on libmesh (UT Austin), which is based on PETsC (Argonne National Laboratories). This is a fantastic example of how open source projects can be built on top of each other to create powerful tools that are tailored to specific applications, with relatively little effort. COBRA is a public repository and community contributions are welcome and encouraged! Developing code for material models can especially extend the capabilities of the application. MOOSE has some built in material models, such as the J2 plasticity model and isotropic power-law hardening, but more complicated models such as Johnson-Cook and combined hardening are better suited to some systems.
