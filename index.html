COBRA

COntact proBlem Restart Application

A MOOSE based application enabling checkpoint simulations
of stochastic contact problems. Built to be used with machine learning
and digital twin models to empower intelligent process monitoring.

  • Learn more

Checkpoint Finite Element Modeling

Modified MOOSE open-source finite element solver used to perform repeated small-scale contact and plasticity simulations. Generated based on process-relevant probability distributions.

  • Learn more

Machine Learning Surrogate Modeling

Machine learning based approach for obtaining finite-element-like results in a fraction of the time. Enabling the development of statistical models for process monitoring and control.

  • Learn more

Integrated Digital Twin Modeling

Fully integrated digital twin models for tracking of process conditions and distributions. Enabling real and accelerated time studies of process outcomes with physics-based results.

  • Learn more

How does it work?

COBRA is an open-source framework for the simulation of systems with stochastic contact problems. COBRA is based existing open-source projects MOOSE and TensorFlow, with a significant amount of process specific code.

Integration with dynamic image analysis particle measurement.

Adaptable to any particle size distribution.

Representative-volume-element sampling of distributed process conditions.
