COBRA

COntact proBlem Restart Application

A MOOSE based application enabling checkpoint simulations
of stochastic contact problems. Built to be used with machine learning
and digital twin models to empower intelligent process monitoring.

  • Learn more

Checkpoint Finite Element Analysis

Modified MOOSE open-source finite element solver used to perform repeated small-scale contact and plasticity simulations. Generated based on process-relevant probability distributions.

  • FEA Background

Machine Learning FEA Surrogate

Machine learning based approach for obtaining finite-element-like results in a fraction of the time. Enabling the development of statistical models for process monitoring and control, as well as integration with digital twin models.

  • ML Approach

The Vision

COBRA is intended to be a tool for the development of digital twin models for process monitoring and control. By enabling the rapid simulation of stochastic contact problems, COBRA can be used to generate training data for machine learning models.
