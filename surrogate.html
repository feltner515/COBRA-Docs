Machine learning surrogate modeling

Finite element models are great, providing repeatable, precise and physics-based predictions. However, they are also computationally expensive to run and time consuming to obtain converged results. The surrogate modeling approach looks to solve this issue, by utilizing FE models to create trainign data for a machine learning model. The goal of the surrogate model is to recreate finite element results, but in a fraction of the time.
